Кремний

Все инструменты для низкоуровневой разработки объединены и связаны в одном приложении — пишите код, редактируйте байты и анализируйте бинарные файлы без дополнительных окон.

Функции

Редактор кода

Пишите и редактируйте низкоуровневый код с поддержкой синтаксиса. Подсветка кода, автодополнение и навигация по коду.

HEX-редактор

Проверка и изменение двоичных данных на уровне байтов (патчинг). Просмотр в шестнадцатеричном и двоичном форматах.

Дизассемблер

Преобразование машинных инструкций в читаемый ассемблерный код. Поддержка различных архитектур процессоров.

Вскоре

Отладчик

Пошаговое выполнение, проверка регистров и памяти. Точки останова, просмотр стека вызовов.

Визуализация памяти

Визуальные карты расположения и распределения памяти. Анализ использования памяти в реальном времени.

Сборка из исходников

Предварительные требования

Зависимость Минимальная версия
CMake 3.16
Qt 6.8.3
Компилятор C++ поддержка C++17

Сборка в Linux

git clone https://github.com/igmunv/cremniy.git
cd cremniy

mkdir build && cd build
cmake ../src
cmake --build .

# Релизная сборка:
cmake ../src -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release

Сборка в Windows

git clone https://github.com/igmunv/cremniy.git
cd cremniy

mkdir build && cd build
cmake -G "MinGW Makefiles" ..\src
cmake --build .

# Релизная сборка:
cmake ..\src -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
